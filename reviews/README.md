# Reviews Archive

Editorial trail for the Council Handbook. Each file is one review pass, exported from the
**Diff Review** workspace (`/review-tools/Council_Handbook_Diff_Review.html`).

## Format
- `YYYY-MM-DD-handbook-diff-<reviewer>.md` — human-readable, renders on GitHub. Each entry:
  a section heading, the quoted line reviewed, a decision (`Accept / Revise / Reject / Discuss`),
  optional tags, and the reviewer's comment.
- `…json` — the exact machine round-trip (`schema: council-review/1`) for re-ingest.

## How a pass works
1. Read the rich diff, hover any line, comment in the margin.
2. **Export Markdown** (and/or JSON) → drop it here and hand it back for the next clean cut.
